# GirlScript Community App Boilerplate
This is a Community App being built for the various GirlScript Chapters that would help the users to know about the latest events that will  be conducted with venue, date and time including the latest programs and many other useful information. The best thing about this Open Source Project is that it would be made available on the Play Store. It is the app "For the community and By the community" and we are really looking forward for your contributions towards this application. Happy Coding !!




## Requirements to contribute

- Basic knowledge of programming is mandatory
-  The ability to use Git and GitHub
- Some knowledge of Dart and how to use Flutter is required for start contributing to the app
- UI/UX Design skills are appreciable but not mandatory
- Some idea about Firebase or willingness to learn it
